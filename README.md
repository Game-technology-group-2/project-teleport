# Game engine #

[![Build Status](https://travis-ci.org/Game-technology-group-2/game-engine.svg?branch=master)](https://travis-ci.org/Game-technology-group-2/game-engine)

A game written in C++, using OpenGL and SDL2, glm and Glew.


## Dependencies ##
* None for now

## Build ##
### GNU/Linux ###
```
mkdir build
cd build
cmake ..
cmake --build .
```
